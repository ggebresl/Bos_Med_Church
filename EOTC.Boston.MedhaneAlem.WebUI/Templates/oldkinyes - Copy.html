ክብር ይእቲ በጌራ ወርቅ ዘጥምቀት-2015
አይቴ ማሰነ ስብእናነ እምነ ዓለም ስቅለት ፥
እመኒ ሐደግነ አፍቅሮ ፀላኢ ሕልፈት ፥
እስመ አበየነ ዮም አፍቅሮ ቢፅነ ትሑት ፥
ወከንቱ ውእቱ ስብእናነ እንበለ ሰላም ፍፅምት ::

ዕጣነ ሞገር በጌራ ወርቅ ዘጥምቀት-2015
ለምንት ይትበሃል ምስጢረ ሥላሴ ምስጢር ፥
ተበሃሉ ሰብእ ማያተ ባህር ፥
ድህረ ሰምኡ ቃለ ፈጣሪ ወተርህዎ ሰማይ መንክር ፥
ትርጓሜ ምሥጢር ወተአምር ፥
እመኒ ተፈትሐ ይእዜ በጥምቀተ ረቢ ማዕምር፥
ወዳግመ አምጣነ ተርእየ በዮሐንስ ገብር ::

አሠረ ንጉሥ በጌራ ወርቅ
በጥምቀት ኮነ ከመ አሐዱ አጋእዝተ ሰማይ ወምድር፥
እንዘ ፈጣረ ይቴሐት በዛቲ እስመ ይከብር ፍጡር ፥
እንበለ ድካም ከመ ይንበር ፥
ወዝንቱ ውእቱ ምሥጢሩ ለፍቅር ፥
ዘየዓቢ እም ኩሉ ነገር ::

ክብር ይእቲ በጌራ ወርቅ ጥቅምት (አቡነ አረጋዊ)-2015
በዘኢይትረአይ ኃይል እሳታዊ ወበዘይትረአይ ኃይል ፥
አመ ሠለስቱ ተንስኡ ለገስፆ አዳም ባዕል ፥
መዊኦ ኢክህሉ እንበለ ነቢብ በቃል ፥
አምጣነ አዳም ተሐብአ ውስተ ግብ ፍቅረ ፈጣሪ ልዑል ::

ዕጣነ ሞገር በጌራ ወርቅ -(አቡነ አረጋዊ)
በምንት ተዓቢ እምነ አዳም አቡከ ፥
አረጋዌ ዳሞ ኃያል ዘተፎኖከ ፥
ውስተ ኤዶም ኢትዮጵያ ለአጥፍኦ ዓርዌ ፀርከ ፥
ፈቃደ ዚአሁ አመ ፈፀምከ ፥
ኵነኔ ወሞት አምጣነ ኢቀርቡ ኃቤከ ፥
ወዳግመ ዓርዌ ገዳማት ውስተ ደብር አምለከ ::

አሠረ ንጉሥ በጌራ ወርቅ
ኦ ካህን ኢይትሌለየ እምግብረ አይሁድ ግብርከ ፥
እስመ ኩለሄ ትሰቅል ወልደ ማርያም እምከ፥
በቤተ መቅደስ ከመ ለመድከ ፥
ወበአሐቲ ትትሌለይ እም ኩሎ ዘኮንከ ፥
በአምልኮ ኃበ ወልድ ኃይልከ ::

ክብር ይእቲ በጌራ ወርቅ ጥቅምት-2015
ወልደ ፀራቢ ተፈድየ ከመ ግብሩ በህገ ብሉይ ሥርዓት ፥
አምጣነ አብዝሐ ግፍዓ ላዕለ ዕፀዋት አግብርት ፥
እንዘ ይብል ፁሩኒ ዲበ ሰረገላ ስቅለት ፥
ወእንበለ ዓምድ ሐነፀ በተ ገሊላ ዘአድለቅለቀ በግብት::

ዕጣነ ሞገር በጌራ ወርቅ ዘመድኃኔዓለም ጥቅምት-2015
አመ ከልሐ ወልድ ማዕከለ ፈያት ክልኤቱ ፥
ፀላዕቱ ለንጊኖስ ወሰራዊቱ
ሐመይዎ እንዘ ይብሉ እግዚአ ላዕሉ ወታህቱ ፥
ኃደጎ በኃጢአቱ
አኮኑ ህብስተ ስኢሎ በውስተ ገዳም በአቱ ፥
ዓርዌ ምድር ተውህቦ ለዛቲ ሥእለቱ ::

አሠረ ንጉሥ በጌራ ወርቅ
ኢይሔይስ ሰብእ እም እንስሳ ለእመ ሞተ በከንቱ ፥
እንበለ ንስሃ ወፍቅር አኮኑ ፀላኤ ርእሱ ውእቱ ፥
ወፈቃደ ልበ ኀበ ምህረቱ ፥
ይግበር ሰብእ እመኒ ይፈቱ ፥
ተወክሎ በእግዚእ ባህቲቱ ::

ክብር ይእቲ በጌራ ወርቅ መጋቢት-2015
እንግዳ ኢኮነ ለድንግል ሞተ አሐዱ ውሉዳ ፥
አምጣነ ድልውት ይእቲ በከመ ነገራ ዘመዳ ፥
ባህቱ ውእቱ ለእመ አምላክ እንግዳ፥
ነፅሮ ውሉዳ ፅሙአ ፍቅረ ሰብእ በቤተ ረድኡ ይሁዳ ::

ዕጣነ ሞገር በጌራ ወርቅ ዘመድኃኔዓለም/ሆሣዕና መጋቢት-2015
መርድዐ ገዻም ጥጦስ አርድእተ ክርስቶስ አሕፍረ ፥
በዋህድ ዕለት እስመ ፈከረ ፥
ብሉየ ወሐዲሰ ምስለ ቅዳሴ ሕቡረ ፥
ወረቢ ድህረ መሀረ፥
ትምህርቶ ኢነስአ ይሁዳ እንዘ ይሔሊ ዲናረ ፥
ወጴጥሮስ ዕብነ አድማስ ኢይሰምእ ወትረ ::

አሠረ ንጉሥ በጌራ ወርቅ
አይቴ ሀለወ ሆሣዕናሁ ለወልደ ዳዊት ዘከብረ ፥
እንዘ በደመና የሐውር ይእዜ እስመ ዲበ አድግ ሖረ ፥
ወበሆሣዕና ድህረ ተአምረ ፥
በንዴት ስቅለት ኃይሎ ተገምረ ፥
ወበሞቱ ኃይለ ሞት ሰበረ ::

ክብር ይእቲ በጌራ ወርቅ ዘበዓለ ትንሣኤ ወ ጰራቅሊጦስ ሰኔ-2016
ሕዝቦ አግዐዘ አግዓዚ እንተ የአምር ኀዘነ ፥
አኮኑ በፅድቅ ዓውደ ቀራንዮ በየነ ፥
ላዕለ ኃጢአት ወሞት ቀተልተ አዳም አቡነ፥
ወበአግዓዚ ተከብሩ ሥላሴ ወደቂቀ አዳም ኩልነ ::፥

ዕጣነ ሞገር በጌራ ወርቅ
ሞተ ሞት አርአየ መምህረ ዘመን ሐዲስ ፥
ለጠበብት ስምዖን ወዮሐንስ :-
እስከ ይብል ሚካኤል አንባቤ ዜና ወርእስ ፥
ኃይለ ጸላኢ ይደመሰስ ፥
አኮኑ ተረክበ ትንሣኤ መድኃኒተ ሥጋ ወነፍስ ፥
ለዘሐመ ኩሉ በዛቲ በለስ ::

አሠረ ንጉሥ በጌራ ወርቅ
መንበረ ልዑል መርዓት ተመነየት ምጽአተ መርዓዊ ውዱስ ፥
አኮኑ አልጸቀት ለጊዜ ከብካብ ሞገስ ፥
በዓለ ሐምሳ ጰራቅሊጦስ ፥
ወዓርከ መርዓዊ ዕንባቆም በዘያሬድ ልብስ ፥
አሠርገዋ ለመርዓት መቅደስ ::

ዋዜማ በጌራ ወርቅ ፡ ዘመድኃኔዓለም
ምድረ ሳሌም ኣሜሪካ በጊዜ ፅጋብ እምነት ዘኢተሰምዓ ተሰምዓ ፥
እስመ አብ ውሉደ ዚአሁ ጊዜ ቀትር በልዓ ፥
ወይቤ ዘመድ መቃብረ ሙሴ ዘፄዓ ፥
ሥጋ አጣሊ ወላህም እፎኑ ተሓጥዓ ፥
በትንሣኤ ኃይለ ሞት ዘሞዓ ::

ክብር ይእቲ በጌራ ወርቅ
ምስለ ክልኤቱ ፈያት ተኈልቈ ወልደ ማርያም ወአብ ፥
አምጣነ ሥጋ ሰረቀ እምዘ ኤልያስ መሶብ ፥
ድህረ ውስተ ምድር ፀንዓ ዓመተ ፍዳ ረሀብ ፥
ወለአዝማዲሁ ያበልእ ሥጋሁ ጊዜ ዕለታ ለዓርብ ::
